# Node.js Express Application Structure

Structure of the app by domains, this makes it easy to manage and scale.

```text
|--src
  |--server.js
  |--customers
     |--controller.js
     |--routes.js
  |--products
     |--controller.js
     |--routes.js
```

## Installation

### Install dependencies
```bash
npm install
```

### Start the application
```bash
npm start
```
